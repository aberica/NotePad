시간제한 : 10초
메모리제한 : 256MB (스택 1MB)
제출횟수 : 20회

32진수는 "0" ~ "9" 와 알파벳 "A"~"V" 를 사용하여 수를 표기하는 방법이다.
알파벳 "A" 부터 "V" 는 각각 10부터 31까지의 값에 대응한다.

길이가 15,000인 두 32진수 A와 B를 전달 받아, (A+B) * |A-B| 를 계산하는 프로그램을 작성하라.

응시자가 서버에 솔루션을 제출했을 때 표시되는 Result 값은 100개의 테스트케이스에 대한 
패널티+수행시간(ms) 이다. 오답인 테스트케이스에 대한 패널티는 1,000,000 이다.
Result 값이 낮을수록 높은 점수를 부여한다.

더 자세한 것은 main.cpp 를 분석하라.

1. 본 검정은 C++ 언어만 지원한다.
2. User Code 내 "malloc.h" 를 제외한 어떠한 헤더파일도 추가할 수 없다.
3. Main 코드는 수정되지 않으며 채점 시 그대로 사용한다.
단, seed 는 변경될 수 있다.
4. 본 검정은 대단히 엄격한 코드 리뷰를 수행한다.

코드 리뷰 시 Visual Studio 와 서버 모두에서 정상적으로 수행되어야 함.
